# Spec: v1.0 Release Infrastructure (IDEA-042/054/055)

**Priority**: P1 | **Est**: 3 weeks

## Why
The framework needs proper release infrastructure before v1.0: documentation site, community channels, and release governance.

## What Changes
1. Documentation site (GitHub Pages or agentharness.dev): Quick Start, API Reference, Examples, Architecture
2. 3 complete examples: Web tool, data processing, code review agent
3. Community: GitHub Discussions, polished README, CONTRIBUTING guide
4. Release process: Beta → RC → v1.0, with feedback collection at each stage
